## What It Does

Pick your skill level (Beginner ‚Üí Pro), optionally filter by stance (Regular, Nollie, Fakie, Switch), and hit **Cook a Combo** to generate a random trick line. Each combo shows:

- Individual tricks with difficulty ratings
- Connectors between tricks
- Total combo difficulty score
- Recent combo history

## Stack

- Vanilla HTML/CSS/JS
- Pre-generated JSON trick database (40 tricks)
- No frameworks, no build step, no database
- Hosted on GitHub Pages

## Data

The trick database (`data/tricks.json`) includes 40 flatground tricks across categories:
- **Basics:** Ollie, Nollie, Caveman
- **Flips:** Kickflip, Heelflip, Tre Flip, Hardflip, Laser Flip, and more
- **Shove-its:** Pop Shove-It, 360 Shove-It, etc.
- **Rotations:** 180s, 360s, Bigspins
- **Balance:** Manual, Nose Manual, Primo Stand
- **Old School:** No Comply, Boneless
